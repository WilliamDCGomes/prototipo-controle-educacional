# Protótipo de Controle Educacional

## Esse aplicativo foi feito na intenção de facilitar e agilizar a troca de informações entre uma instituição de ensino e o estudante.
## Com ele, um aluno é capaz de acompanhar suas notas e falta, trabalhos, aulas, avisos e requisitar algumas coisas diretamente pelo aplicativo.

### Tela de Introdução

#### A primeira do aplicativo é uma tela de introdução, onde é falado um pouco sobre o aplicativo em si.

### Tela de Login

#### Logo em seguida temos a tela de Login, onde o aluno irá se autenticar no aplicativo.

![Tela de Login](https://github.com/WilliamDCGomes/projeto-tcc/blob/main/ReadmeFiles/Tela%20de%20Login.jpg)

### Fluxo de Cadastro

#### Caso seja o primeiro acesso, o aluno pode solicitar um registro, pelo fluxo de cadastro, onde após aprovado pela instituição, será fornecido um RA para ele logar no aplicativo.

![Fluxo de Cadastro](https://github.com/WilliamDCGomes/projeto-tcc/blob/main/ReadmeFiles/Tela%20de%20Cadastro.jpg)

### Dashboard

#### Uma vez logado no sistema, o aluno terá acesso a um Dashboard contendo várias informações e ações relacionadas ao seu curso.
